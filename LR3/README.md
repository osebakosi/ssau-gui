# PyQt + SQLite Application

Приложение на PyQt6 с использованием SQLite для работы с базой данных.

## Описание

Приложение демонстрирует работу с SQLite базой данных через графический интерфейс PyQt6:

- **Меню** с функциями подключения и отключения от БД
- **Кнопки** для выполнения различных SQL запросов
- **QComboBox** для выбора колонок из таблицы
- **QTabWidget** для отображения результатов запросов в разных вкладках

## Функционал

### Меню
- **Set connection** - устанавливает соединение с БД, создает тестовые данные
- **Close connection** - закрывает соединение, очищает все вкладки

### Кнопки
- **bt1** - выполняет `SELECT name FROM sqlite_master`, результат в Tab2
- **bt2** - выполняет `SELECT * FROM users`, результат в Tab4
- **bt3** - выполняет `SELECT * FROM products`, результат в Tab5

### QComboBox
- Заполняется именами колонок из таблицы `users` после подключения
- При выборе колонки выполняется запрос и результат отображается в Tab3

### Вкладки
- **Tab1** - `SELECT * FROM sqlite_master` (создается при подключении)
- **Tab2** - результат запроса bt1
- **Tab3** - результат выбора из QComboBox
- **Tab4** - результат запроса bt2
- **Tab5** - результат запроса bt3

## Установка

1. Установите Python 3.8 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Тестовые данные

При первом подключении автоматически создаются две таблицы:

### Таблица users
- id (INTEGER)
- name (TEXT)
- email (TEXT)
- age (INTEGER)

### Таблица products
- id (INTEGER)
- product_name (TEXT)
- price (REAL)
- category (TEXT)

## Использование

1. Запустите приложение
2. Выберите в меню **Menu → Set connection**
3. После подключения станут активны все кнопки и ComboBox
4. Нажимайте на кнопки для выполнения запросов
5. Выбирайте колонки в ComboBox для просмотра данных
6. Для отключения выберите **Menu → Close connection**

## Требования

- Python 3.8+
- PyQt6 6.10.0
- SQLite3
